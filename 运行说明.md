# 运行说明

## 1. 环境要求

- **操作系统**：Windows / macOS / Linux（不限）
- **Python版本**：3.8 或以上
- **必需库**：
  - pandas（数据处理）
  - numpy（数值计算）
  - scikit-learn（机器学习模型与评估）
  - xgboost（XGBoost模型）
  - matplotlib（绘图）
  - openpyxl（读取 .xls 文件）

## 2. 安装依赖

在终端（命令行）中执行以下命令安装所有依赖：

```bash
pip install pandas numpy scikit-learn xgboost matplotlib openpyxl
```

若使用国内镜像源加速，可添加 `-i https://pypi.tuna.tsinghua.edu.cn/simple`

## 3. 文件准备

确保以下两个文件位于同一文件夹下：
- main.py（上述完整代码）
- 面板数据.xls（您提供的原始数据文件）

## 4. 运行代码

在终端中进入该文件夹，执行：

```bash
python main.py
```

或直接在IDE（如PyCharm、VSCode）中打开并运行该脚本。

## 5. 预期输出

### 控制台打印
- 数据加载信息、样本量
- 训练/验证/测试集划分数量
- 各模型在测试集上的 MSE、RMSE、MAE、R²（应复现论文结果：线性回归 R²≈0.544，随机森林 R²≈0.827，XGBoost R²≈0.879）
- 前10个重要特征及其得分

### 生成文件
- feature_importance.png：特征重要性条形图（保存于当前目录）

## 6. 常见问题排查

| 问题 | 解决方法 |
|------|----------|
| ModuleNotFoundError: No module named 'openpyxl' | 执行 `pip install openpyxl` |
| 数据文件读取失败 | 确认文件名是否为 `面板数据.xls`，且与 `main.py` 在同一目录 |
| XGBoost 安装失败（Windows） | 可尝试 `pip install xgboost --upgrade`，或从官方文档获取预编译包 |
| 结果数值与论文有微小差异 | 由于交叉验证分割的随机性，RMSE/R²可能存在 ±0.002 的浮动，属正常现象（已固定 random_state=42 以保证最大可复现性） |

## 补充说明（关于时间划分与滞后项）

代码中构造了一阶滞后项 `税负水平_lag1`，因此 2000年的数据不进入训练集（用于生成2001年的滞后值）。

实际建模样本为 2001—2024年，其中：
- 训练集：2001—2017（527条）
- 验证集：2018—2020（93条）
- 测试集：2021—2024（124条）
